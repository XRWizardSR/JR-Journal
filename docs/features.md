# ✨ Platform Features

JR Journal is domain-specific to the healthcare environment, designed to understand residency workflows, medical logging formats, and shift-based scheduling. 

### Core Capabilities

* **🗣️ Natural Language Queries:** Allows JRs to interact conversationally. (e.g., Ask *"How many samples did I test this month?"* for instant reporting).
* **📅 Automated Roster Syncing:** Parses and categorizes complex medical duty types (Academic Rounds, Cultural events, Weekend shifts, Night duty) and syncs them automatically.
* **📊 Clinical Data Extraction:** Converts unstructured natural language (e.g., *"Completed Ward 38 HIC rounds"*) into highly structured, audit-ready database rows.
* **🤖 Multi-Agent Orchestration:** Intelligently delegates tasks so the user doesn't have to navigate different apps or menus. The Primary Agent handles the routing to specialized Schedule and Logbook sub-agents.
* **🔌 MCP Tool Integration:** Seamless read/write access to external productivity tools (Google Calendar) and enterprise-grade databases (AlloyDB).
